하이패스 이중결제(오납) 알림을 받았다면, **먼저 어느 주체(하이패스 등록사/카드사/톨게이트 운영사)가 결제한 건인지 확인한 뒤 해당 창구에 환불을 신청**하면 됩니다. 확인 기준일: 2026-07-21.

이중결제는 크게 두 경우로 나뉘는데요: 하이패스 단말기와 카드 모두에서 결제된 경우(중복 청구) 또는 시스템 오류로 동일한 결제가 두 번 발생한 경우입니다. 각각 처리 주체와 필요한 서류가 다를 수 있으니 **결제 명세(카드내역)와 통행기록(통과시간·톨게이트 위치)**을 준비해서 바로 확인하는 게 빠릅니다.

처음에는 헷갈릴 수 있지만 아래 절차를 따라 하면 환불 창구를 찾고 신청하는 데 도움이 됩니다.

## 환불 신청의 기본 원칙

- **누가 결제했는지 찾기**: 카드사 명세서, 하이패스 단말기 등록사(예: 하이패스 서비스 제공사) 또는 고속도로 톨게이트 운영사 중 실제 결제 주체 확인.
- **증빙 준비**: 카드명세서(이중결제 내역), 차량번호·통과시간, 하이패스 단말기 등록정보(카드·충전사 등).
- **신청 창구**: 결제 주체(카드사) → 또는 하이패스 등록사 / 한국도로공사 고객센터 → 필요 시 톨게이트 운영사 방문.
- **처리 소요**: 보통 며칠에서 수주까지 다양(카드사 환불 처리 주기, 사업자 확인 필요). **정확한 소요는 신청 창구에서 다시 확인**.

---

## 어디에 환불을 신청해야 하나요?

확인해보니 실제로는 세 주체 중 하나가 환불을 처리합니다.

- 카드사: 카드명세에 이중결제가 찍혔다면 우선 카드사에 환불 요청 가능. 카드사가 결제 취소(차지백) 또는 환불을 진행.
- 하이패스 등록사(또는 고속도로 운영사): 하이패스 전용 충전·결제사업자에서 결제가 이뤄진 경우 사업자에서 환불 처리.
- 톨게이트 운영사(지역별): 특정 톨게이트 운영사 시스템 오류로 이중청구 시 운영사에서 환불 조치.

**중요: 환불 창구는 '결제 주체'에 따라 달라지므로 카드사 명세와 하이패스 화면(또는 앱)에서 결제처를 먼저 확인하세요.**

## 실제 환불 신청 절차(순서별 안내)

1. 카드사·하이패스 앱에서 결제 내역 확인
   - 카드사 앱·명세서에서 동일한 결제(같은 금액·시간)가 두 건인지 확인.
   - 하이패스 앱이나 등록 사업자 사이트에서 통행 내역(차량번호, 통과시간)을 확인.

2. 증빙자료 확보
   - 카드사 결제내역 캡처(이중결제 표시)
   - 통행 영수증 또는 통행시간·톨게이트 명시된 기록
   - 차량번호, 하이패스 단말기 등록증(영수증) 또는 충전 내역

3. 창구별 신청
   - 카드사: 카드사 고객센터 또는 앱의 부정거래/이의신청 메뉴에서 접수
   - 하이패스 등록사 또는 한국도로공사: 고객센터 문의(온라인 민원 또는 전화)로 환불 요청
   - 톨게이트 운영사 방문(필요 시): 직접 증빙을 들고 현장 방문해 접수

4. 처리 확인 및 추적
   - 접수번호·민원번호 받기
   - 처리 예상기간과 환불 방법(계좌환불/카드취소) 확인

## 준비서류 및 제출 방식

- 기본 준비물
  - 본인 확인용 신분증(대리 신청 시 위임장)
  - 카드사 결제내역(캡처 또는 명세서)
  - 통행 영수증 또는 통행시간/톨게이트 정보
  - 하이패스 기기 등록 증빙(예: 구입 영수증, 등록 인증서) — 필요할 때

- 제출 방식
  - 온라인(하이패스 등록사 사이트 · 카드사 앱) 첨부
  - 이메일 또는 팩스(사업자별 허용 시)
  - 콜센터 접수 후 우편/방문 제출 요구가 있는지 확인

## 창구별 비교표(어디서 확인하고 무엇을 놓치기 쉬운지)

| 확인 항목 | 어디서 확인                    | 놓치기 쉬운 점                   | 바뀔 수 있는 부분        |
| --------- | ------------------------------ | -------------------------------- | ------------------------ |
| 결제 주체 | 카드명세·하이패스 앱           | 같은 시각 다른 주체 결제 혼동    | 사업자 통합(플랫폼 변경) |
| 신청 창구 | 카드사 / 한국도로공사 / 등록사 | 잘못된 창구에 접수하면 처리 지연 | 처리위임·전산연계 변경   |
| 증빙서류  | 앱 캡처·영수증                 | 영수증 미보관                    | 전자문서 제출 방식 변경  |

처음에는 헷갈릴 수 있지만, 위 표처럼 '무엇을 어디서 확인할지'를 먼저 정리하면 신청이 빨라집니다.

## 처리 기간과 환불 방식(현재 기준으로는)

- 처리 기간: **일반적으로 접수 후 며칠에서 수주(7~30영업일)**까지 소요될 수 있습니다. 다만 카드사 취소는 비교적 빠른 편이고, 사업자 확인이 필요하면 더 길어질 수 있습니다.
- 환불 방식: 카드결제 취소(원결제 취소) 또는 계좌입금(사업자 환불). **카드사 결제취소인 경우 카드사 명세에서 '취소' 처리 확인 필요**.
- 확인해보니 일부 사업자는 환불 시점·정책을 내부 규정으로 운영하므로 **접수 시 예상 소요와 환불 방식**을 반드시 문의하세요.

## 흔히 발생하는 예외 및 주의사항

- 동일 카드사가 아닌데 결제내역상 두 건(예: 카드사 결제 + 하이패스 선불충전)인 경우 처리 주체가 달라질 수 있음.
- 대리인 신청 시 위임장과 신분증 필요.
- 환불 처리 후 카드취소(크레딧)로 표시되더라도 실제 카드사 반영 시점은 카드사 정책에 따름.
- 고속도로 통행료 지급 체계가 지역별로 다를 수 있으니, 톨게이트 운영사가 별도인 경우 운영사 확인이 필요.

## 공식 확인 경로(바로 확인할 곳)

- 하이패스 등록사(단말기·충전사) 공식 홈페이지 고객센터(로그인 후 통행·충전내역 확인)
- 카드사 고객센터(앱/웹의 결제내역·부정거래 이의신청 메뉴)
- 한국도로공사 또는 해당 톨게이트 운영사 고객센터(홈페이지 민원/조회)
- **확인 팁**: 영수증·결제명세의 '결제일시'와 하이패스 통행시간을 비교하면 주체 확인이 쉬워집니다.

## 시간이 지나면 바뀔 수 있는 항목

아래 항목들은 제도·시스템·사업자 정책에 따라 바뀔 수 있으니 접수 전 꼭 확인하세요.

- 환불 처리 기간(카드사·사업자별 내부 정책 변경 가능)
- 온라인 접수 경로(앱·웹 통합 여부)
- 제출 가능한 전자증빙 형식(스캔·캡처 허용 여부)
- 환불 방식(계좌입금 허용 여부, 대행 처리 등)

## 지금 바로 확인할 3가지

1. 카드 명세와 하이패스 앱에서 **결제 주체와 결제시간**을 비교한다.
2. 증빙(카드내역·통행시간·차량번호)을 캡처해서 보관한다.
3. 결제 주체(카드사 또는 하이패스 등록사) 고객센터에 접수하고 **접수번호**를 꼭 받는다.

## 공식 페이지에서 다시 봐야 할 항목

- 환불 신청 가능 기간(예: 신고 기한)
- 제출 가능한 증빙 자료 목록
- 대리인 신청 시 필요한 서류 및 위임 양식
- 처리 예상기간과 환불 수단

## Q&A (자주 묻는 질문)

Q: 하이패스 알림에 '오납'이라고 뜨면 무조건 환불 가능한가요?  
A: 대부분의 경우 환불 대상이지만, **결제 주체·중복 유형**에 따라 절차가 달라집니다. 카드사·사업자 확인 후 조치됩니다.

Q: 카드사와 하이패스 등록사 중 어디에 먼저 연락해야 하나요?  
A: 카드명세에 이미 청구가 올라와 있다면 카드사에 먼저 문의해 '중복청구' 여부를 확인하고, 필요 시 하이패스 등록사에 추가 증빙을 요청하세요.

Q: 환불 소요가 길면 어떻게 추적하나요?  
A: 접수번호로 고객센터에 추적 요청하거나, 온라인 민원 조회를 통해 처리상태를 확인하세요.

Q: 대리인 신청은 가능한가요?  
A: 가능합니다. 다만 위임장(서명 포함)과 대리인 신분증을 요구하는 경우가 많으니 사전에 확인하세요.

## 확인 체크리스트

- [ ] 카드명세서에서 이중결제 내역 캡처하기
- [ ] 하이패스 통행기록(시간·톨게이트) 캡처하기
- [ ] 결제 주체(카드사/등록사) 확인하기
- [ ] 해당 창구(카드사·하이패스 등록사·운영사)에 접수하기(접수번호 수령)
- [ ] 접수 후 처리 예상 기간과 환불 방식 확인하기
- [ ] 환불 완료 시 카드명세/계좌 입금 여부 확인하기

마지막으로, 가장 빠른 해결은 "결제 주체를 정확히 확인하고 그쪽 고객센터에 증빙을 바로 제출하는 것"입니다. 접수할 때는 **접수번호와 담당자 확인**을 꼭 받아두면 추후 처리 추적이 수월합니다.
